# Algorithm Interviewer - Prompts Reference

本文档提供各数据结构、算法类型的经典引导话术。使用时根据具体题目灵活调整。

---

## 数据结构引导话术

### 数组 (Array)

**特征识别**：
- "输入是序列数据吗？"
- "需要随机访问吗？"
- "数据是连续存储的吗？"

**优化方向**：
- "需要频繁在中间插入/删除吗？考虑链表"
- "需要快速查找吗？考虑排序后二分，或用哈希表"
- "有重复元素吗？如何去重？"

**边界提醒**：
- "空数组怎么处理？"
- "单元素数组呢？"
- "索引越界问题考虑了吗？"

---

### 链表 (LinkedList)

**特征识别**：
- "需要频繁在头部插入/删除吗？"
- "数据大小不固定吗？"
- "不需要随机访问吗？"

**技巧提示**：
- "考虑过用 dummy node 简化边界处理吗？"
- "需要双指针（快慢指针）吗？"
- "能不能反转链表来简化问题？"

**边界提醒**：
- "空链表怎么处理？"
- "单节点链表呢？"
- "注意不要丢失 next 指针"

---

### 栈 (Stack)

**技巧提示**：
- "单调栈可以解决问题吗？"
- "可以用栈模拟递归吗？"

**特征识别**：
- "需要先进先出 (FIFO) 吗？"
- "需要层次遍历吗？"

**技巧提示**：
- "双端队列 (deque) 能不能解决问题？"

## 算法类型引导话术

### 双指针

**适用场景提示**：
- "数组有序吗？可以用首尾指针吗？"
- "需要找子数组吗？可以用滑动窗口吗？"
- "需要链表中点吗？用快慢指针？"

**模板**：
```cpp
// 首尾指针
int left = 0, right = n - 1;
while (left < right) {
    int sum = nums[left] + nums[right];
    if (sum == target) return {left, right};
    else if (sum < target) left++;
    else right--;
}

// 快慢指针
ListNode* slow = head, *fast = head;
while (fast && fast->next) {
    slow = slow->next;
    fast = fast->next->next;
}
```

---

### 滑动窗口

**适用场景提示**：
- "需要找最长/最短子数组吗？"
- "需要满足特定条件的子数组吗？"

**关键问题**：
- "窗口什么时候扩大？什么时候缩小？"
- "怎么判断窗口满足条件？"
- "需要记录窗口内的什么信息？"

---

### 二分查找

**适用场景提示**：
- "数组有序吗？"
- "可以排序后使用吗？"
- "有单调性吗？"

**关键问题**：
- "二分的条件是什么？"
- "left、right 怎么更新？"
- "循环结束条件是什么？"

**常见错误提示**：
- "注意 mid 的计算避免溢出：mid = left + (right - left) / 2"
- "注意边界：left <= right 还是 left < right？"

---

### 排序

**适用场景提示**：
- "需要排序才能解决问题吗？"
- "用什么排序算法？时间复杂度？"

**技巧提示**：
- "可以用排序后双指针吗？"
- "可以用桶排序/计数排序吗？"
- "需要自定义比较函数吗？"

---

### 动态规划 (DP)

**特征识别**：
- "问题可以分解为重叠子问题吗？"
- "有最优子结构吗？"
- "需要求最值/方案数/可行性吗？"

**引导步骤**：
1. **定义状态**："dp[i] 表示什么？"
2. **状态转移**："怎么从 dp[i-1] 推出 dp[i]？"
3. **初始化**："dp[0] 是多少？"
4. **遍历顺序**："先遍历什么，后遍历什么？"

**优化方向**：
- "可以空间优化吗？只用前几个状态？"
- "需要二维 DP 吗？"

---

### 回溯 (Backtracking)

**适用场景提示**：
- "需要找出所有解吗？"
- "需要枚举所有可能性吗？"
- "组合、排列、子集问题？"

**模板**：
```cpp
void backtrack(路径, 选择列表) {
    if (满足结束条件) {
        result.add(路径);
        return;
    }
    for (选择 : 选择列表) {
        做选择;
        backtrack(路径, 选择列表);
        撤销选择;
    }
}
```

**关键问题**：
- "路径是什么？选择列表是什么？"
- "什么时候结束？"
- "需要剪枝吗？"

---

### 贪心

**特征识别**：
- "局部最优能导致全局最优吗？"
- "有单调性吗？"

**引导问题**：
- "每一步最优选择是什么？"
- "为什么局部最优能导出全局最优？"
- "反例是什么？"

---

### BFS/DFS

**BFS 适用场景**：
- "需要最短路径？"
- "需要层次遍历？"
- "图的最短边数路径？"

**DFS 适用场景**：
- "需要遍历所有路径？"
- "需要判断连通性？"
- "回溯/搜索？"

**关键问题**：
- "怎么避免重复访问？visited 数组？"
- "BFS 的队列里存什么？"
- "DFS 递归终止条件是什么？"

## 调试技巧 Checklist

### 代码审查阶段

**逻辑检查**：
- [ ] 边界条件：空、单元素、极值
- [ ] 初始化是否正确
- [ ] 循环条件是否正确（<= 还是 <）
- [ ] 变量更新位置是否正确
- [ ] 是否有整数溢出风险

**脑内模拟步骤**：
1. "用最简单的例子走一遍"
2. "用边界例子走一遍"
3. "每一步的变量值是什么？"
4. "循环什么时候结束？"

---

### 测试用例设计

**正常用例**：
- 题目给的示例
- 自己构造的简单例子

**边界用例**：
- 空数组/空字符串
- 单元素
- 全相同元素
- 极值（INT_MAX、INT_MIN）

**错误用例**：
- 无解的情况
- 负数
- 重复元素

---

### 调试信息添加

**打印策略**：
```cpp
// 循环中
for (int i = 0; i < n; i++) {
    printf("i=%d, nums[i]=%d\n", i, nums[i]);
}

// 递归中
void dfs(int x) {
    printf("Enter dfs: x=%d\n", x);
    // ...
    printf("Exit dfs: x=%d\n", x);
}

// 关键变量
printf("hash_map size=%d, key=%d, value=%d\n",
       (int)mp.size(), key, mp[key]);
```

**对比预期**：
```
预期输出：...
实际输出：...
第 X 步开始不一致
```

## 常见错误引导

### 数组越界
**提示**："注意数组索引从 0 开始，访问 nums[n] 会越界。"

### 死循环
**提示**："循环的更新条件写了吗？left++/right++？"

### 重复计算
**提示**："可以用 memoization 或 DP 优化吗？"

### 整数溢出
**提示**："相加可能超过 INT_MAX，考虑用 long long 或先判断。"

### 忘记排序
**提示**："算法需要有序输入吗？先排序试试。"
